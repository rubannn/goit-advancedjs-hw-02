## goit-advancedjs-hw-02

### How start local

1. Install npm
   ```bash
   npm install
   ```

2. Fix conflict
   ```bash
   npm audit fix --force
   ```

3. Run in dev-mode
   ```bash
   npm run dev
   ```

4. If need install package
   ```bash
   npm install flatpickr
   npm install izitoast --save
   ```
